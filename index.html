🤖 AI Prompt — How to Generate a Patch File
Copy the prompt below and send it to any AI (Claude, ChatGPT, etc.) along with your lecture list. The AI will produce a .json file you can upload directly using ↑ Patch (for one subject) or ↑ Full (for the whole course).
What each file type is
↑ Patch — Adds or replaces a single subject. Safe: only touches that one subject, everything else stays.
↑ Full — Replaces the entire curriculum at once. Use when you have all subjects ready.
